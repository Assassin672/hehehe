LIMBAJ DE PROGRAMARE C++
C++ este un limbaj de programare de nivel înalt, dezvoltat de Bjarne Stroustrup la începutul anilor 1980, ca o extensie a limbajului C.
Este un limbaj orientat pe obiecte, dar suportă și programarea procedurală. C++ este folosit pentru dezvoltarea de software performant, jocuri, aplicații pentru sisteme încorporate și software de mare viteză.
Oferă control asupra resurselor hardware și permite optimizări avansate. De asemenea, C++ include concepte precum moștenirea, polimorfismul și încapsularea, fiind un limbaj flexibil și puternic.
